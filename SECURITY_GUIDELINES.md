# 보안 가이드라인

## 노출 금지 항목

- 회사 내부 소스코드 (의사 코드 또는 구조만 표현)
- API 키, 토큰, 시크릿 (Supabase, Jira, AWS 등)
- 내부 URL (Jira, Jenkins, 어드민 페이지, S3 버킷명, CloudFront 배포 ID)
- 사용자 데이터, 개인정보가 포함된 스크린샷
- 비즈니스 지표 원본 (매출, 정확한 비용 금액, 내부 KPI)

## 안전한 표현 방법

| 위험 | 대체 |
|------|------|
| 실제 코드 | 의사 코드 또는 구조만 보여주는 간소화 코드 |
| 실제 스크린샷 | 마스킹 처리 또는 목업/와이어프레임 재제작 |
| 정확한 비용 금액 | 비율/퍼센트로만 표현 (82% 감소, 94% 경량화) |
| 내부 URL | example.com 또는 제거 |
| 사용자 수 | 이미 공개된 정보(앱스토어 등)만 사용 |

## 커밋 전 체크

- [ ] 이미지에서 EXIF 메타데이터가 제거되었는가?
- [ ] 스크린샷에 개인정보/내부 URL이 노출되지 않는가?
- [ ] NDA 위반 사항이 없는가?
